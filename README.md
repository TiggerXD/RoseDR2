RoseDR Dataset Classes

ชุดข้อมูล RoseDR ประกอบด้วยภาพใบกุหลาบที่ได้รับผลกระทบจากโรคและศัตรูพืชชนิดต่าง ๆ

รายละเอียดของแต่ละคลาส

โรคจุดดำ (Black Spot)

โรคที่เกิดจากเชื้อรา ทำให้เกิดจุดสีดำลักษณะกลมหรือไม่สม่ำเสมอบนใบกุหลาบ ส่งผลให้ใบเหลืองและร่วงก่อนกำหนด หากปล่อยไว้เป็นเวลานานอาจทำให้ต้นอ่อนแอและเจริญเติบโตได้ไม่ดี

โรคราน้ำค้าง (Downy Mildew)

โรคที่เกิดจากเชื้อราน้ำ ทำให้เกิดรอยแผลสีม่วง แดง หรือน้ำตาลบนใบ และอาจมีลักษณะคล้ายคราบเชื้อราที่ด้านล่างของใบ โรคนี้สามารถทำให้ใบร่วงและลดความแข็งแรงของต้นกุหลาบได้

โรคราแป้ง (Powdery Mildew)

โรคที่เกิดจากเชื้อรา มีลักษณะเด่นเป็นผงสีขาวคล้ายแป้งปกคลุมบนใบ ลำต้น หรือดอกกุหลาบ ส่งผลให้การเจริญเติบโตของพืชชะงักงันและทำให้ใบผิดรูปได้

ไรแดง (Red Mites)

ศัตรูพืชขนาดเล็กที่ดูดกินน้ำเลี้ยงจากใบกุหลาบ ทำให้ใบมีจุดสีเหลือง ซีด หรือแห้งกรอบ เมื่อมีการระบาดรุนแรงอาจส่งผลให้ใบร่วงและทำให้ต้นกุหลาบอ่อนแอลง
